# IBM Web Development Course Project

This repository contains a practice project created while completing the course  
**"Introduction to Web Development with HTML, CSS, JavaScript"** offered by IBM on Coursera.

The goal of this project was to practice basic frontend web development concepts such as building a webpage structure, styling with CSS, and adding simple interactivity using JavaScript.

## Live Demo
🔗 https://sneh777a.github.io/ibm-web-dev-course-project/

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Personal portfolio style layout
- Navigation menu with sections
- Skills section
- Projects section
- Recommendation section
- Basic interactive forms using JavaScript

## Purpose of the Project
This project was built as part of my learning journey in web development.  
It helped me understand how different frontend technologies work together to create a simple website.
